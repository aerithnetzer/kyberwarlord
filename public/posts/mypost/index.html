The Utter Banality of Kyberspace

15.03.2024 00:00

The internet has become small. It has become undeniably boring. I no longer engage with the internet, I merely view. Did we ever engage with the internet at all? Is it possible to engage with the internet at all? Even this website, something that I built myself, relies on code running somewhere on another server. Is this really my website at all?
